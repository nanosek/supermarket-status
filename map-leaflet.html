This is a mashup prototype for a crowd sourcing "waiting time before a supermarket" map - it is functional, but obviously of not in production quality. It was created as a fun/learning project as part of the #WirvsVirus hackathon in 2020. There is no real server backend for this solution - this page is using the public available MQTT Server mqtt.eclipse.org to publish your reported status and the nice Overpass API to get the list of supermarkets from the OpenStreetmap data.
